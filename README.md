# 🔐 Password Manager (React + LocalStorage)

A simple and secure Password Manager built using React.

## 🚀 Features

- Save passwords
- Edit passwords
- Delete passwords
- Copy to clipboard
- Show / hide password
- Data stored in LocalStorage
- Toast notifications
- Unique ID using UUID

## 🛠️ Tech Used

- React
- JavaScript
- Tailwind CSS
- React Toastify
- UUID
- LocalStorage
